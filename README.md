## THISOZ YOUTUBE Project.
---
- 함수형 컴포넌트와 React Hook을 통한 상태관리
- Youtube API, Postman 을 통한 API 
  * 검색기능(search), 영상정보(videos), 채널정보(channels)
- PostCSS 를 통한 CSS 모듈화
- .env 파일을 이용한 환경 변수 설정 (API Key 암호화)
